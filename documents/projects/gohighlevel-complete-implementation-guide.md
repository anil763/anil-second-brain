## Workflow Automation Principles

### 1. Every Lead Gets Nurtured Automatically
- No manual follow-up required
- 7-14 day email sequences trigger on form submission
- SMS for hot leads only (high-intent signals)

### 2. Tag-Based Segmentation
- Every contact gets tagged based on source and behavior
- Tags control which sequences they enter
- Clean lists = better deliverability

### 3. Ascension Model
- Each funnel has a clear upsell path
- Automation handles upsell offers at optimal timing
- Cross-sell between funnels for qualified contacts

### 4. Metrics-Driven Optimization
- Track opens, clicks, conversions per email
- A/B test subject lines
- Monthly review and optimization

- SMS: "Reminder: Our call is tomorrow at [TIME]. Looking forward to it!"

- SMS: "Call starting in 1 hour. Here's your link: [ZOOM LINK]"

### Post-Call Workflow:

**Trigger:** Call completed (marked in calendar)

**Actions:**
1. Send proposal email (manual trigger with template)
2. Wait 2 days → Follow-up if no response
3. Wait 5 days → Final follow-up
4. If no response after 7 days → Move to nurture

# SECTION 8: WORKFLOW SETUP INSTRUCTIONS

## How to Set Up Workflows in GoHighLevel

### Step 1: Access Automation Builder

1. Log into your GHL subaccount
2. Click **Automation** in the left sidebar
3. Click **Workflows** (or **Automation** depending on version)
4. Click **+ Create Workflow**

### Step 2: Choose Trigger

**For Form Submission:**
1. Click **Add Trigger**
2. Select **Form Submitted**
3. Choose the specific form from dropdown
4. Save

**For Purchase:**
1. Click **Add Trigger**
2. Select **Payment Received** or **Order Submitted**
3. Filter by product if needed
4. Save

**For Tag Added:**
1. Click **Add Trigger**
2. Select **Contact Tag Added**
3. Choose the tag
4. Save

### Step 3: Add Actions

**To Send Email:**
1. Click **+** after trigger
2. Select **Send Email**
3. Choose template OR write new
4. Set "From" name and email
5. Save

**To Add Tag:**
1. Click **+**
2. Select **Add Contact Tag**
3. Enter tag name (will create if doesn't exist)
4. Save

**To Wait:**
1. Click **+**
2. Select **Wait**
3. Choose duration (e.g., "1 day")
4. Save

**To Add to Pipeline:**
1. Click **+**
2. Select **Add to Pipeline**
3. Choose pipeline and stage
4. Save

### Step 4: Add Conditional Logic

**If/Else Branch:**
1. Click **+**
2. Select **If/Else**
3. Set condition (e.g., "Contact has tag X")
4. Add actions under YES branch
5. Add actions under NO branch

**Common Conditions:**
- Contact has tag: [tag name]
- Contact clicked link in email
- Contact opened email
- Contact purchased product
- Days since last activity

### Step 5: Save and Activate

1. Name your workflow clearly (e.g., "UGC-Lead-Nurture-Sequence")
2. Click **Save**
3. Toggle **Active** to ON
4. Test with a dummy contact

- Emoji vs. no emoji
- Personalization vs. generic

- Long vs. short
- Story vs. direct

- P.S. line vs. no P.S.

- CTA button text
- Testimonial placement

4. Send Version B to second half
5. Wait 48 hours
6. Check metrics
7. Use winner for future sends
