## HOW TO USE THESE

**Rotation Strategy:**
- Week 1: Use Pinned Comment #1 + Verbal Script #1
- Week 2: Use Pinned Comment #2 + Verbal Script #2
- Week 3: Use Pinned Comment #3 + Verbal Script #3
- Week 4: Use Pinned Comment #4 + Verbal Script #4
- Week 5: Use Pinned Comment #5 + Verbal Script #5
- Week 6+: Rotate back through, or mix/match

**Why rotate:**
- Keeps it fresh for repeat viewers
- Tests which messaging converts best
- Feels authentic, not robotic
- Algorithm prefers varied content

**Key Elements in EVERY script:**
✅ Price ($97)
✅ What they get (20-min custom audio)
✅ How fast (2-3 days)
✅ Scarcity (5/week limit)
✅ Direct link mention (bio/pinned)
✅ Emotional hook (transformation, clarity, understanding yourself)
