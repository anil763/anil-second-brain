# 2nd Brain

A beautiful personal knowledge base that captures conversations, concepts, and insights.

## Features

- 📚 **Document Viewer** — Beautiful markdown rendering with syntax highlighting
- 📁 **Folder Organization** — Concepts, daily journals, projects, decisions
- 🔍 **Search** — Find docs across your knowledge base
- 🔗 **Linked References** — See how concepts connect
- 📝 **Daily Journals** — Auto-generated summaries of your work sessions
- ✨ **Obsidian + Linear Vibes** — Clean, focused interface

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Markdown:** remark + rehype
- **Icons:** Lucide React
- **Storage:** Local filesystem (`documents/` folder)

## Getting Started

### 1. Install Dependencies

```bash
cd projects/2nd-brain
npm install
```

### 2. Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### 3. Add Documents

Place markdown files in the `documents/` folder and update `metadata.json`:

```json
{
  "id": "my-doc-id",
  "title": "My Document",
  "path": "folder/my-doc.md",
  "folder": "folder",
  "type": "concept",
  "preview": "Short preview text",
  "tags": ["tag1", "tag2"]
}
```

## Document Workflow

### Creating a Concept Doc

```markdown
# My Concept

## Definition
Clear explanation of the idea.

## Why It Matters
Impact and relevance.

## Related
- [[Other Concept]]
- [[Another Concept]]

## Created
2026-02-07
```

### Creating a Daily Journal

```markdown
# Daily Journal - 2026-02-07

## Summary
High-level overview of the day.

## Topics Discussed
- Topic 1
- Topic 2

## Key Decisions
- Decision 1
- Decision 2

## Tomorrow
- [ ] Next step 1
- [ ] Next step 2

## Mood
😊 How you're feeling
```
